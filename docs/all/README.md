# easyj-all

`easyj-all` 是由其他所有基础模块合并而成的一个模块，如果要使用所有基础模块，就直接引用此依赖吧：

```xml
<dependency>
    <groupId>icu.easyj</groupId>
    <artifactId>easyj-all</artifactId>
    <version>0.7.5</version>
</dependency>
```

基础模块的详细文档编写中...
* core：核心
* config：配置
* crypto：数据库模块
* data：数据操作
* db：数据库
* jwt：JSON Web Token
* login：登录
* poi：Office操作
* redis：操作Redis
* web：数据库模块
