# easyj-maven-plugin


### 功能：

`easyj-maven-plugin`是EasyJ社区自研插件，提供了以下几个功能:

1. [simplify-pom](maven-plugin/simplify-pom.md)：统一版本号、简化POM。
2. [spring-boot-extend](maven-plugin/spring-boot-extend)：`spring-boot-maven-plugin`扩展插件。


### 建议

请使用最新版本的该插件，功能更完善！
