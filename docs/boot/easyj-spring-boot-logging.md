# easyj-spring-boot-logging

`easyj-spring-boot-logging`提供了快速集成logback，并提供快速集成推送日志到Logback（集成ELK）、推送日志到Kafka的appender。

并且，将logback的xml配置，转换为yaml配置，使大家更方便的启停各种appender。
